E2EE Extension

Encrypts on Send — matches CipherKey keyboard

Off = emoji wrappers (same as keyboard emoji mode)

Must match your Android keyboard passphrase to decrypt.

Shows 🔓 real text under encrypted messages in chat.

Setup for your friend (receiver)
  1. Install this same extension in Chrome.
  2. Open the extension → enter the same passphrase as you → Save.
  3. Turn on Decrypt incoming.
  4. Open web.whatsapp.com — under the symbol message they’ll see 🔓 and your real words.

WhatsApp still stores the encrypted symbols; only their browser shows the decrypted line.
